================================================================================
                          QUICK START GUIDE
================================================================================

1. Install Dependencies:
   npm install

2. Set Up Environment:
   cp .env.example .env.local
   # Add Supabase credentials

3. Set Up Database:
   - Create Supabase project
   - Run SQL scripts from sql/ folder
   - Configure RLS policies

4. Start Development:
   npm run dev

5. Build for Production:
   npm run build

6. Deploy:
   npm i -g vercel
   vercel --prod
